# V8: 研发费用加计扣除政策、研发投入调整与企业创新效率 — 聚焦分析

## 概述

V8 将研究主线从"政策是否增加创新数量"调整为**"政策是否影响研发投入调整与相对创新效率"**。
在 V5 14方向探索性筛选的基础上, V8 围绕创新效率主线进行完整的实证分析,
包括数据审计、变量构造、基准DID、研发投入调整、创新效率主模型、效率拆解、
三类异质性、政策暴露强度、稳健性检验、图表生成和最终报告。

## 研究逻辑

```
创新数量 DID 不显著 → 政策效果不体现为专利数量扩张
        ↓
研发投入 DID 显著为负 → 制造业相对非制造业研发投入增长较慢
        ↓
创新效率 DID 显著为正 → 单位研发投入创新效率相对提升
        ↓
DID(效率) ≈ DID(创新产出) − DID(研发投入) → 效率提升来自投入端结构调整
```

## 数据

### 输入
- `data/firm_panel_v4.csv` — v4 企业面板 (CSMAR 8表合并 + 省级数据, 46,705 obs × 5,834 firms)

### 基准样本
- 2017-2022: 26,772 obs × 5,404 firms (62.5% 制造业)
- stock_code-year 唯一: ✓
- 每企业最多 6 条观测

### 关键变量处理
| 步骤 | 说明 |
|------|------|
| 比率转换 | rd_intensity, rd_staff_ratio 从百分比(0-100)转换为比率(0-1) (÷100) |
| 对数变换 | ln_rd_expense_10k = ln(1 + rd_expense/10000), 万元口径 |
| 效率构造 | eff = ln(1+专利) − ln(1+研发投入), 对数差分型 |
| 比率效率 | 分母 ≤ 0 → NaN, 1%/99% 缩尾后计算 asinh |
| 缩尾 | 仅对新建比率型效率变量进行 1%/99% 缩尾 |

## 核心实证结果

### 1. 创新数量 (简略呈现, 全部不显著)

| 因变量 | DID系数 | p值 |
|--------|---------|-----|
| ln(发明专利申请) | -0.025 | 0.315 |
| ln(发明专利授权) | -0.011 | 0.622 |
| ln(专利总申请) | -0.002 | 0.959 |
| ln(专利总授权) | +0.035 | 0.330 |

**结论**: 政策效果并不主要体现为专利数量扩张。构成研究转向依据。

### 2. 研发投入调整 (重点呈现, 显著为负)

| 因变量 | DID系数 | p值 |
|--------|---------|-----|
| ln(研发支出, 万元) | **-0.288** | <0.001*** |
| 研发强度 (0-1) | **-0.003** | 0.025** |
| ln(研发人员) | **-0.197** | <0.001*** |
| 研发人员占比 | +0.000 | 0.951 |

**解释**: 制造业研发投入在政策后绝对水平上升, 但相对非制造业增长较慢。

### 3. 创新效率 (核心展示, 全部高度显著)

| 效率指标 | DID系数 | p值 |
|----------|---------|-----|
| eff(发明申请/研发支出) | **+0.264** | <0.001*** |
| eff(发明授权/研发支出) | **+0.277** | <0.001*** |
| eff(发明申请/研发人员) | **+0.172** | <0.001*** |
| eff(发明授权/研发人员) | **+0.186** | <0.001*** |

### 4. 效率拆解恒等式

DID(效率) ≈ DID(创新产出) − DID(研发投入) = (-0.025) − (-0.288) = **+0.264** ✓

效率提升主要来自研发投入相对增长较慢, 而非专利数量显著增加。

### 5. 异质性分析

| 异质性 | 关键发现 |
|--------|---------|
| 高研发基础 | 交互项 +0.225*** (效率改善集中于研发基础强的企业) |
| 制造业内部 | high_pre_rd_post2021 +0.198*** |
| 所有制 | 国企组 +0.270***, 非国企组系数更大但不稳定 |
| 政策暴露强度 | policy_exposure 系数显著为正 |

### 6. 稳健性

| 检验 | 结果 |
|------|------|
| 效率指标口径 (万元/百万元) | 系数稳定, 仅常数项变化 |
| 强固定效应 (+Prov×Year) | 系数保持稳定 (+0.255*** vs +0.264***) |
| 事件研究 (基准=2020) | 创新产出平行趋势成立; 研发投入存在政策前差异 |
| 安慰剂 (2019/2020) | 效率指标假想DID显著 (反映政策前趋势收敛) |

### ⚠ 方法论警示

事件研究显示: 政策前 (2017-2019) 制造业研发支出系统性高于非制造业
(event_2017: +0.687***, event_2018: +0.396***, event_2019: +0.296***),
政策后差距收敛至零附近。效率DID部分反映政策前趋势收敛,
**非严格的因果效应**。全文将结果表述为"相对效率改善", 不声称政策因果效应。

## 关键约束

- 不得编造或隐藏结果
- 不显著的创新数量结果必须简略报告, 不得删除
- 效率结果表述为"相对效率改善", 不得写成"政策显著增加创新数量"
- 事件研究和安慰剂检验的负面结果需透明呈现
- 平行趋势问题必须在方法论讨论中说明
- `lev` (资产负债率) 不可用, 使用 `cashflow_ratio` 作为补充控制
- 2023年全行业100%加计扣除稀释了制造业定向激励的识别
