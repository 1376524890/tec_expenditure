### 当前可做分析

**基准DID**:
```
patent_stock_it = beta * manufacturing_post2021_it 
                + gamma * X_it + delta_i + lambda_t + epsilon_it
```

**含机制检验**:
```
rd_intensity_it = beta * manufacturing_post2021_it + ...  (创新投入机制)
patent_stock_it = beta * manufacturing_post2021_it 
                + gamma * gov_subsidy_rd_it + ...  (补贴机制)
```

**含省级交互**:
```
patent_stock_it = beta * manufacturing_post2021_it 
                + gamma * manufacturing_post2021_it * sci_tech_exp_ratio_pt + ...
```
