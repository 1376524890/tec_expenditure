# 科技自主创新政策实证研究 — v4 完整报告

## 一、研究设计

### 1.1 政策背景

2021年制造业研发费用加计扣除比例从 **75% → 100%**（财税〔2021〕13号），非制造业维持75%。2023年起全行业统一提高至100%（财税〔2022〕28号）。

### 1.2 识别策略

**双重差分（DID）**：处理组 = 制造业企业（`manufacturing=1`），对照组 = 非制造业企业。政策冲击时点 = 2021年。估计方程：

```
Y_it = α + β × (manufacturing_i × post2021_t) + γ × X_it + μ_i + λ_t + ε_it
```

- 企业固定效应（μ_i）吸收不随时间变化的企业特征
- 年份固定效应（λ_t）吸收共同时间趋势
- 标准误按企业层面聚类
- 引擎：`linearmodels.PanelOLS`（within-transformation 吸收 FE）

### 1.3 v4 相对 v3 的扩展

| 维度 | v3 | v4 |
|------|----|----|
| 基准样本 | 2017-2022 | **2017-2024** |
| 省级财政交互 | 无 | **新增**（DID × 省财政科技支出强度等5组） |
| 省R&D经费交互 | 无 | **新增** |
| PPML | 有（statsmodels GLM + dummies, 不可行） | 跳过（线上不可用，建议Stata/R补充） |

## 三、实证结果

### 3.1 基准DID

| 样本 | 系数 | 标准误 | p值 | 显著性 | N |
|------|------|--------|-----|--------|----|
| **2017-2024 (v4)** | **-0.0409** | 0.0221 | 0.064 | * | 37,694 |
| 2017-2022 (v3) | -0.0238 | 0.0234 | 0.310 | | 26,770 |
| 2016-2022 | -0.0232 | 0.0232 | 0.316 | | 30,205 |

**解读**：2021年制造业100%加计扣除政策对发明专利申请未观测到统计显著的促进作用。系数方向为负，且2017-2024扩展样本中在10%水平边缘显著，表明延长窗口后制造业相对创新产出可能进一步下降，与2023年起全行业统一提高至100%的稀释效应一致。

**加入post2023控制后**：`manufacturing_post2021` 系数不变（`post2023` 在PanelOLS中被时间FE完全吸收），无法单独识别post2023效应。

### 3.2 省级财政科技支出交互（v4核心新增）

| 模型 | 关键变量 | 系数 | p值 | N |
|------|----------|------|-----|----|
| **P1**: DID × 省财政科技支出占比 | `manufacturing_post2021` | -0.0487 | 0.279 | |
| | `did_x_prov_sci_tech` | 0.0006 | **0.947** | 36,569 |
| **P2**: DID × 省R&D强度 | `manufacturing_post2021` | -0.0323 | 0.529 | |
| | `did_x_prov_rd_intensity` | -0.0042 | **0.800** | 35,636 |
| **P3**: DID × ln(省财政科技支出) | `manufacturing_post2021` | -0.0875 | 0.401 | |
| | `did_x_ln_prov_sci` | 0.0070 | **0.688** | 36,569 |
| **P4**: 三重差分（高vs低财支省份） | `manufacturing_post2021` | -0.0357 | 0.196 | |
| | `did_x_high_sci_prov` | -0.0096 | **0.762** | 37,694 |
| **P5a**: 高财政科技支出省份子样本 | `manufacturing_post2021` | -0.0534 | 0.121 | 20,249 |
| **P5b**: 低财政科技支出省份子样本 | `manufacturing_post2021` | -0.0207 | 0.519 | 17,445 |

**解读**：所有省级财政科技支出交互项均不显著（p > 0.5）。高财政科技支出省份DID系数绝对值更大（-0.053 vs -0.021），但差异不具统计显著性。地方财政科技投入未观测到对加计扣除政策创新效应的显著调节作用。

| **研发投入强度** | rd_intensity | 2017-2024 | -0.192 | 0.129 | |
| | | 2017-2022 | **-0.285** | 0.025 | ** |
| | | 2016-2022 | **-0.288** | 0.026 | ** |
| **研发人员** | ln_rd_staff | 2017-2024 | -0.073 | 0.086 | * |
| | | 2017-2022 | **-0.138** | 0.001 | *** |
| | | 2016-2022 | **-0.160** | 0.000 | *** |
| **补贴渠道** | ln_invention_apply | 2017-2024 | -0.040 | 0.068 | * |
| | | 2017-2022 | -0.023 | 0.337 | |
| **政策暴露强度** | ln_invention_apply | 2017-2024 | -0.044 | 0.096 | * |
| | | 2017-2022 | -0.022 | 0.422 | |

**解读**：机制检验显示，政策后制造业企业的**研发强度显著下降约0.29个百分点**（2017-2022，p=0.025），**研发人员对数化后显著下降约14%**（p=0.001）。这一结果与政策预期方向相反，可能原因：
1. 加计扣除降低税负后，企业可能减少账面研发投入强度（与营业收入的相对变化有关）
2. 非制造业（对照组）同期研发投入增长更快
3. 2021-2022年疫情冲击对制造业的影响更为严重

`policy_exposure`（政策暴露强度 = 2017-2020平均研发强度 × manufacturing × post2021）系数为正且显著（p=0.037），说明政策前研发强度越高的制造业企业，政策后发明专利申请相对更多——存在研发基础差异。

## 四、结论

### 4.1 主要发现

1. **基准DID不显著**：2021年制造业研发费用加计扣除比例提高至100%对发明专利申请**未观测到统计显著的促进作用**。所有样本、所有替代因变量的DID系数均不显著。

2. **省级财政科技支出无调节效应**：DID × 省财政科技支出占比、DID × 省R&D强度等5组交互项全部不显著，地方财政科技投入未观测到对政策效应的显著调节。

3. **研发投入反而下降**：政策后制造业的研发强度（-0.29pp，p=0.025）和研发人员（-14%，p=0.001）显著下降，与政策预期方向相反。

4. **平行趋势通过**：事件研究显示政策前系数不显著，安慰剂检验也通过，DID识别框架基本有效。

5. **2024年显著恶化**：事件研究中2024年系数显著为负（p=0.046），与全行业统一提高至100%后制造业相对优势消失一致。

### 4.2 可能的解释

- **政策效应有限**：加计扣除是事后税收优惠（需先有研发支出才能享受），对企业创新决策的边际激励可能弱于直接补贴
- **疫情冲击**：2021-2022年新冠疫情对制造业冲击可能大于非制造业（供应链中断、需求萎缩）
- **对照组改善**：非制造业（尤其是信息技术服务业）同期创新增长可能更快
- **政策提前反应**：部分企业可能在2020年已预期政策而提前增加研发
- **价格效应**：研发投入的名义值变化可能受到成本上升（人力、设备）的影响

### 4.3 研究局限

1. `tax_saving_est` 是估算值（RDSpendSum × rd_deduction_rate × 0.25），非真实税务数据
2. `lev`（资产负债率）不可得，用 `cashflow_ratio` 替代控制
3. PPML模型线上不可用（需Stata ppmlhdfe或R fixest::fepois）
4. 2024年 `province_sci_tech_exp` 为2023年填充值
5. 2023年起全行业100%加计扣除稀释了制造业处理组的相对优势
6. 不能声称严格因果识别 — 事件研究提供平行趋势证据但无法完全验证
