# 数据补充需求清单

## 优先级分类

### 必须补（最低补数清单 Top 10）

| 排名 | 必补数据 | 字段名 | 原因 |
|------|----------|--------|------|
| 1 | 真实研发费用 | `rd_expense` | 机制检验必须有真实研发投入 |
| 2 | 当年发明专利申请数 | `invention_apply` | 最适合作为高质量创新主因变量 |
| 3 | 当年发明专利授权数 | `invention_grant` | 衡量高质量创新结果 |
| 4 | 2017-2018 年企业数据 | `year` | 增强平行趋势检验 |
| 5 | 企业注册地省份 | `province` | 匹配地方财政科技支出 |
| 6 | 省级财政科学技术支出 | `province_sci_tech_exp` | 支撑财政科技支出分析 |
| 7 | 省级一般公共预算支出 | `province_budget_exp` | 计算财政科技支出占比 |
| 8 | 政府补助总额 | `gov_subsidy` | 检验财政补贴机制 |
| 9 | 所得税费用 | `income_tax_expense` | 计算企业实际税负 |
| 10 | 利润总额 | `profit_before_tax` | 计算实际所得税率 |

| 必须 | 企业基础信息 | 企业注册地省份 | `province` | 匹配省级财政科技支出 | CSMAR、Wind、年报公司基本信息 |

## 建议数据结构

| 数据表 | 主键 | 内容 |
| --- | --- | --- |
| 企业年度主表 | `stock_code + year` | 企业财务、研发、补贴、行业、地区 |
| 专利年度表 | `stock_code + year` 或 `firm_name + year` | 专利申请、授权、发明专利、专利质量 |
| 省级年度表 | `province + year` | 财政科技支出、GDP、R&D 强度、知识产权保护 |
| 政策变量表 | `industry_code + year` 或 `province + year` | 加计扣除比例、地方政策、政策冲击变量 |
| 政府采购表 | `firm_name + year` | 政府采购合同、采购金额、采购次数 |

## 公开数据可自动采集部分

以下数据可通过自动化浏览器从公开来源采集：

1. **省级财政科学技术支出** — 国家统计局 (stats.gov.cn)、中国科技统计年鉴
2. **省级一般公共预算支出** — 财政部、各省财政厅
3. **省级 GDP** — 国家统计局、中国统计年鉴
4. **技术市场成交额** — 中国科技统计年鉴
5. **研发费用加计扣除比例政策文件** — 财政部、税务总局官网
6. **知识产权保护指数** — 市场化指数报告
